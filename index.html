API Chatbot

This project is written in Java and uses the libraries: gson and pircbot. In this project pircbot is used to implement a chatbot while gson is used to parse json files. Postman was used for API implementation. The chatbot will connect to freenode.net to channel #osmang and connects to 2 APIs. The weather API and uselessfacts API which will return the weather of a zipcode or city while the uselessfacts will give an interesting fact.

Collatz Conjecture

Basic Collatz Conjecture Analysis Using Java For now it only iterates up to 100,000. Collatz Conjecture states that any number greater than or equal to 2 will eventually be reduced to 1 when using the following set of rules: When the number is even implement n/2 and when the number is odd implement 3n+1, where n is our number.
